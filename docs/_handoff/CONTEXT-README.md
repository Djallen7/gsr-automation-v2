# GSR Automation v2 - Course bundle: contents

## How to use the artifact
1. Work each module in order. Tab 1 (Blueprint) captures your intent before the lesson. Tab 2 (Learn) teaches it with a quiz. Tab 3 (Decide) locks a build decision.
2. "Ask the Consultant" on any module copies a context-loaded prompt to your clipboard; paste it into a Claude chat and add your question.
3. At the Finale, Export Build Plan. Open Claude Code in gsr-automation-v2, paste the plan, and have it build module by module.

## Rules that never bend
- NO em dashes anywhere in copy (Daniel's preference).
- The David Rule: automation talks to cloud APIs only; never the live broadcast chain, ProPresenter, the QNAP NAS, ATEM, Companion, or Tailscale.
- Build in gsr-automation-v2 (not gsr-blueprint).
