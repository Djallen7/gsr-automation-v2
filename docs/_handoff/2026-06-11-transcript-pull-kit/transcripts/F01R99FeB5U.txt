A lot of us use Claude Code every day,
but once the session is over, it becomes
surprisingly hard to inspect what
actually happened. You might remember a
workflow went well or a session messed
up or that Claude used a lot of tools,
but after that most of the context stays
buried inside the local file and it is
very hard to explore them properly. So,
I built CC Lens, an open-source
dashboard that turns your Claude Code
data into something that you can
actually explore. It reads everything
from your dot Claude folder and gives
you a clear view of your project,
session, model, activity patterns, and
overall usages. You can browse all of
your session, filter it by signals like
web, MCP, agent workflows, and quickly
find the session that is worth looking
at. The setup is pretty simple. All you
need to do is just run NPX CC Lens and
it will open the dashboard in your
browser.
